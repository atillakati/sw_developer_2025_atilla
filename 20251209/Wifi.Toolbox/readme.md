# Wifi.Toolbox.Tools

Die Wifi.Toolbox.Tools enthält unsere grundlegenden Methoden, welche wir während der Veranstaltung geschrieben haben und für dir Umsetzung von Applikationen in der Console sehr hilfreich sind. Jeder Student ist angehalten dieses Package selbstständig zu pflegen. Die Notwendige Infrastruktur wird im Unterricht vorbereitet.

Folgende Methoden sind im Package enthalten:
- Einlesen von numerischen Werten
- Methoden zur Gestaltung von Console-Oberflächen
